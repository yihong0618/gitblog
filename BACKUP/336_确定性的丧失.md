# [确定性的丧失](https://github.com/yihong0618/gitblog/issues/336)

今天想找一些不用 LLM 生成但是每天可以自动发出来的好玩的东西，可悲的是除了 LeetCode easy 我找不到任何可以每天手写一些代码并且能让我每天多少有一些期待感并且好玩的东西。更可悲的是我找不到了，用 LLM 找了半天依旧毫无收获，并且我已经把条件放宽到不用写代码，任何好玩的东西都行。。。可惜毫无收获

## 确定性的丧失

如标题所云，突然有种确定性丧失的悲哀，我想做一些 LLM 以外的东西，最后却只能用 LLM 去查找，但是毫无收获，wiki 添加完了，每日一首诗歌几年了，AI 生成的图片视频 what ever 厌倦了，只想用去找一些确定性的东西 but nothing, 2023 年形成了一道分界线，分界着人类的时代，人类和 LLM 共存的时代，以及大部分是 LLM 生成的时代。

## Good old days

那个考研大概率会找到更好的起薪，那个努力刷题就能进大厂，那个八股文面试背的好也许就能进入 BAT 的时代。但是大人，时代变了，变得猝不及防，现在已经不是讨论 LLM 会不会取代程序员的问题了，LLM 会取代大部分东西，程序员又算什么呢？

## 上海见了朋友

大概只有自己当前的酒量是确定的，精确的把握着自己大概能继续喝多少，感觉差不多了，结账再分别。

## 年年有今日

岁岁有今朝，人生大部分都是不如意的，未来一定会更好也只是骗局，只有当下的举杯，说出那句俗的不能再俗的祝酒词才是真的把时光留住。
“年年有今日，岁岁有今朝”，干杯

> “那个考研大概率会找到更好的起薪，那个努力刷题就能进大厂，那个八股文面试背的好也许就能进入 BAT 的时代。”
> 
> 这种确定性，看起来都是在描述“赚钱的方式”的确定性。
> 
> 我的理解是，如果生活的重心放在赚钱上。而且是这种通过竞争（打赢其他人类对手）来赚钱。那么在ai时代，发现ai也在和人类做竞争，把ai视作一种潜在的竞争对手，自己和ai比较发现自己卷不赢它，这种失落感起名叫做确定性的丧失。
> 
> 感觉有点微妙。
> 
> 但是人生很明显不是只有赚钱。ai无法代替你享受生活。我们还是按照原来的方式呼吸吃饭睡觉。
> 
> ai的出现是否会垄断赚钱的途径，让大家活不下去我觉得存疑。
> 
> 别人已经做过的事情，我们在可以在做一遍。别人能够做到事情。我们做不到或者要花很久。
> 这不是什么很难为情的事情。
> 我们在过去都有很多次类似的经历。
> 比我们强的人很多。现在加了ai。这种因为对此而产生的失落感其实并不陌生。
> 
> 就像大哲学家广井菊理曾经讲过：“你的敌人并不在这里”。
