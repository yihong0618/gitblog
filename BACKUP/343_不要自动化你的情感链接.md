# [不要自动化你的情感链接](https://github.com/yihong0618/gitblog/issues/343)

最近收到很多类似的邮件少部分我会回一下，这些一看就是用 Claw 等 Agents 自动发的邮件，尴尬的笑一下的同时也在思考，我，我们该用 AI 自动化什么，不自动化什么。

<img width="1143" height="124" alt="Image" src="https://github.com/user-attachments/assets/9239259f-63c1-42d8-ad2f-0ece1f72324a" />

- 邮件，这种情感链接的还是不自动化的好，自动化的邮件，其实和垃圾邮件没有本质的区别
- 写给自己的文章不要自动化，否则未来再读，他和任何一篇 LLM 生成的文字没区别
- commit message 和 PR message 不要自动化（这条仅仅针对我自己）
- 给朋友打的文字，聊天不要自动化，否则我也跟 LLM bot 没有区别
- 读书不要自动化
- 学习好玩的东西不要自动化

LLM helps me enjoy life not take my whole life
maybe
so do you.

.....这种应该会越来越多不可逆的。
